# viron-doc

## How to Develop

```
$ cd website
$ npm install
$ npm run start
```

then access to [here]( http://localhost:3000).

## How to Publish

```
$ cd website
$ GIT_USER=<GIT_USER> USE_SSH=true npm run publish-gh-pages
```

`GIT_USER`: The username for a GitHub account that has commit access to this repo.
